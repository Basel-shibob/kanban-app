# Kanban App

A full-stack project management app built with Node.js, Express, MongoDB, and Next.js.

## Live Demo
- Frontend: https://kanban-app-sigma-gray.vercel.app
- Backend: https://kanban-app-production-88c3.up.railway.app

## Features
- User authentication with JWT
- Create and manage boards
- Create tasks with To Do / In Progress / Done columns
- Move tasks between columns
- Protected routes

## Tech Stack
**Frontend:** Next.js, React, Tailwind CSS  
**Backend:** Node.js, Express.js  
**Database:** MongoDB Atlas  
**Auth:** JWT + bcrypt  
**Deployment:** Vercel (frontend), Railway (backend)

## Running Locally

### Backend
cd server
npm install
npm run dev

### Frontend
cd client
npm install
npm run dev
